#  Data Analytics Project

##  Overview
This project demonstrates an end-to-end **data analytics workflow**, starting from raw data loading to insight generation and visualization.  
The analysis focuses on extracting meaningful business insights using **Python, SQL, and Power BI**, following industry-standard data analytics practices.

---

##  Dataset
- Source: Structured customer transaction dataset
- Format: CSV
- Data includes:
  - Customer demographics
  - Purchase details
  - Transaction history
  - Ratings and subscription information
- Missing and inconsistent values were handled during data cleaning

---

##  Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **MySQL Workbench**
- **Power BI**
- **Jupyter Notebook**
- **SQL**

---

## Project Workflow / Steps

### 1️ Data Loading
- Imported dataset into Python using Pandas
- Performed initial inspection using `.info()` and `.describe()`

### 2️ Exploratory Data Analysis (EDA)
- Analyzed distributions, trends, and patterns
- Identified missing values and outliers
- Visualized key metrics using charts and graphs

### 3️ Data Cleaning
- Handled missing values using appropriate techniques
- Standardized column names
- Removed redundant or unnecessary columns
- Ensured data consistency for analysis

### 4️ SQL Analysis (MySQL Workbench)
- Loaded cleaned data into MySQL
- Executed SQL queries to answer business questions such as:
  - Revenue analysis
  - Customer segmentation
  - Product performance
  - Subscription and repeat purchase behavior

### 5️ Power BI Dashboard
- Built an interactive dashboard to visualize insights
- Included filters and KPIs for better decision-making
- Highlighted trends across customers, products, and revenue

### 6️ Report Creation
- Compiled findings into a structured report
- Included insights, charts, and business recommendations

---

##  Dashboard
The Power BI dashboard provides:
- Key performance indicators (KPIs)
- Revenue and sales trends
- Customer segmentation analysis
- Product and category-level insights

---

##  Results & Insights
- Identified high-performing products and customer segments
- Analyzed impact of discounts and subscriptions on revenue
- Discovered purchasing trends across different demographics
- Provided actionable insights for business optimization
